# ThelowStorageMod

Minecraft 1.8.9 Forge環境において、ストレージ（倉庫）の管理を劇的に効率化するGUIオーバーレイModです。
TheLow倉庫の中身を瞬時に検索し、一画面で全ての倉庫へアクセスすることを可能にします。

## 🌟 主な機能

* **全ストレージ一括プレビュー**: ページ分けされた倉庫の中身を、現在の画面にグリッド状に展開して表示します。
* **高速アイテム検索**: 画面上部の検索バーから、全倉庫内のアイテムをリアルタイムでフィルタリング。ヒットしたアイテムを持つ倉庫は赤枠で強調されます。
* **シームレスな操作**: Mod GUIから直接アイテムを取り出したり、設定したキーでホットバーと入れ替えたりできます。
* **インテリジェント・スケーリング**: 解像度に合わせてGUIを自動調整し、プレイヤーインベントリを常に画面下部に固定。作業効率を落としません。
* **自動キャッシュ**: 一度開いた倉庫の内容はローカルにJSON形式で保存され、次回から開かなくても中身を確認できます。

## 🛠 使い方

### 1. 倉庫データの登録
Modを導入後、サーバーの倉庫メニューからストレージを開くと、その中身が自動的にキャッシュされます。
> [!TIP]
> 初回起動の際は、Mod GUIを一時的に **OFF** にして、すべてのストレージを一度ずつ開いてください。
> Mod GUIは画面右上のボタンからON/OFFの切り替えが可能です。

### 2. GUIの操作
チェスト画面を開くと、独自のModオーバーレイがバニラの画面を覆う形で表示されます。

* **検索バー**: 画面上部をクリック、またはフォーカスしてアイテム名を入力します。
* **スクロール**: マウスホイール、または画面右側のスクロールバーで上下に移動できます。
* **アイテム操作**:
    * **左クリック**: 通常のアイテム移動。
    * **Shift + 左クリック**: インベントリと倉庫間の高速移動。
    * **ホットバー指定キー**: マウスオーバーしているアイテムを、対応するスロットと入れ替え（バニラのキー設定に準拠）。
    * **ドロップキー (Q)**: マウスオーバーしているアイテムをその場に捨てます（`Ctrl+Q` でスタックドロップ）。

### 3. Mod設定とレイアウトのカスタマイズ
画面右上のボタン、またはForgeの設定画面から調整が可能です。

* **Mod GUI: ON/OFF**: Modオーバーレイの有効/無効を切り替えます。
* **Layout**: 1行に表示する倉庫の列数（3〜5列）を変更できます。

## 📋 システム要件
* **Minecraft**: 1.8.9
* **Minecraft Forge**: 1.8.9 対応版

## ⚠️ 注意事項
* **エンダーチェスト**: バニラの仕様に基づき、上部3段分（27スロット）のみ操作可能です。下段の空白エリアは誤操作防止のためマスク処理（黒塗り）されます。
* **ページ認識機能**: Modが自動的にページボタン（「nページ目を表示」等）を認識してページ遷移をサポートします。
* **表示ページ**: 現在サーバー側で開いているページに対応する倉庫エントリは、緑色の枠でハイライトされます。

キャッシュデータは以下のディレクトリに `倉庫名.json` として保存されます。
`.minecraft/config/ThelowStorage/`
